A Spring Boot REST API backend for an e-commerce application(e-shop) with product catalog, shopping cart, and order management.

Browse and manage products with detailed information.
Add/remove items, calculate totals.
REST API: Full CRUD operations with hypermedia-driven design.
Spring Data JPA ORM with H2 database.
Lombok for boilerplate reduction, exception handling.

Prerequisites:
Java 11+ & Maven

Run the Application
mvn clean install
mvn spring-boot:run
